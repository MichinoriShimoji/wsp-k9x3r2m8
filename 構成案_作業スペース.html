文体・設計の方針

本編の声色
「である」調。感情語を書かず、具体物と読点の切り方で情感を伝える。
コラムの声色
本編よりやや柔らかく、一人称の手触りを強める。散文的に、呼吸を作る。
ユーモア
抑制的。章に1〜2回、軽い転調として配置。
硬派要素
各章に「読者が一つ賢くなる硬い塊」を必ず置く(学問用語、方法論、統計、事例など)。
章の組み立て
具体的シーン(入口)→ 方法論・歴史(深掘り)→ 次章への問い(閉じ)。
構造の勘所
前半で世界と学問を提示、第3〜4章で距離を詰め、第5〜6章で論理を畳みかけ、第7章で再帰し、終章で社会へ射程を開く。
本編
序章 12p
言語が減ることって、問題ですか?
タイトルの問いを正面から提示。自己紹介、問いの来歴、本書の方針。
著者紹介(言語学者、伊良部島方言を15年以上記録) → タイトルの問いは何十回も投げかけられてきた問い → 世界の言語状況の統計 → 本書の方針(多角的に問いを解きほぐす) → 章立ての予告。
第1章 18p
言語学者は、何を見ているのか
立ち位置の紹介。ユーモアを解禁してトーンを定める。
言語学 / 日本語史 / 国語学 / 生成文法 / 認知言語学 / フィールド言語学の棲み分け。方言周圏論、「ポケモン言語学」批判、消滅危機方言学の誕生。「エアコンの効いた部屋」対「エアコンのない小屋」の対比。
第2章白眉 35p
伊良部島へ — 神話者に出会うまで
フィールドワークと神話者列伝。読み物としての中心。
パラオ語から琉球諸語への転向、オーストラリア国立大、伊良部島を選んだ理由(父方の故郷という動機を軽く添える)、6ヶ月住み込みと自作辞書、話者との出会い方、神話者の定義、揺れの問題、調査プロセス(基礎語彙調査・面接・自然談話)。※失敗譚(「雨に降られた」・例文倫理)はコラム2へ移動。本章はフィールドワークの成功譚に集中する。
第3章 22p
消滅の現場 — 最後の一人がいなくなる日
トーンを落として、事実と歴史のスケールで読者を打つ章。
ワッポ語(ローラ・サマーサル, 1990)、ボ語(2010)、Nuu-chah-nulth(アルバン・マイケル, 2016)の訃報 → 統計(世界6,000〜7,000言語、今世紀中に半減〜90%消滅、上位300言語で人口の95%) → 消滅のメカニズム(威信言語と現地語、家庭での言語選択、進学・就職での断絶) → 日本の危機言語(アイヌ語、西表島船浮方言) → 明和の大津波、東日本大震災 → 記録するという営み(Boasian trilogy)。
第4章 22p
話者たちは、何を失うのか
当事者の感情を正面から扱う章。
スティグマの歴史、方言札、なぜ話者たちは子供に言葉を渡さなかったのか → 言語は単なる道具ではない(帰省で方言にホッとする感覚、海外で日本語に安心する経験) → 話者が感じる苦しみ、バイリンガル化の強制 → ハワイ語・マオリ語の復興 → 琉球で復興が起きにくい理由、しまくとぅばの日の限界 → 書き下ろし話者インタビューセクション。
第5章 18p
「役に立つから守れ」というロジックの罠
功利主義批判。分析的なハイライト。
『Wired』特集の語り(薬草知識、ハヌノオ族の40の土の表現) → この論法のキャッチーさと潜む罠 → 「役に立つから守れ」は言語を殺してきたロジックそのもの → 明治の標準語政策、沖縄の方言撲滅運動はすべて「役に立つ」で遂行された → バイオ・プロスペクティングの倫理問題。
第6章 18p
あなたは、日本語を手放しますか?
読者へのブーメラン。本書の転調点。
九大授業の2つの問い(①少数言語の消滅は問題か → 半々 ②日本語を英語にするか → 誰も賛成しない) → 学生の反応の矛盾 → 「なぜ君は日本語を手放さなくていいのに、少数言語話者は少数言語を手放してもいいのか?」 → 日本語の部分的消滅(シラバス英語化、全英語授業) → 少数言語話者の嫌悪感・窮屈さはこの比ではない。
第7章note原稿を統合 22p
私の母語 — ウチナーヤマトグチ
note「私の母語について」(2024/2025)を本編に統合。ウチナーヤマトグチ話者としての立場表明と、復興運動のパターナリズム批判。
冒頭:第6章の読者への問いかけを受けて、「では、自分自身はどうか」と問い直す(書き下ろし) / ウチナーヤマトグチとは何か(Level 0〜3 の連続体、接触言語/クレオロイド) / 証拠性(シヨッタ vs シタ)、「今雷鳴りよったよ」「なんか触りよった」「甘くなりよった」、反実仮想への転用 / ニックネーム音韻論(真美→まーみー、慎太郎→しんたー) / なぜ誰も研究しないのか — 伝統的琉球諸語の消滅危機が重大すぎて後回しにされてきた / 「失った」ことにされる構造 — 復興運動の文脈でウチナーヤマトグチ話者は「いない」ことにされる / 「『自分たちの』言語はここにある、まずこれを認めなければならない」 / 伝統方言の記録保存と、自分の母語への愛着の両立 / 結尾:終章への橋渡し(書き下ろし)。※note原稿の構造と段落は基本維持。本書の流れに合わせて導入と結尾のみ書き下ろし。
終章 15p
言語が減ることと、私たちの社会の論理
序章の問いに応答。社会全体の論理へ射程を開く。
序章の問いに戻る → 「問題だ、だから守ろう」も「消えるものは消える」も取らない理由 → 記録するという営みの意味 → 射程の拡張:多数派の論理は難民・移民・障害者・性的少数者・女性・非正規雇用者などに向かう論理と同じ構造 → 「あなたは今、弱者ではないかもしれない。しかし、どの文脈で弱者になるかは誰にもわからない」 → 問いを読者に手渡して閉じる。
コラム「あわい」
1あわい 4p / 第2章後
神話者列伝
第2章の「神話者」を具体的な人物肖像で受ける。
伊良部・椎葉・都城で出会った3人の神話者。職業、出会いの経緯、印象的なエピソード。感情語を書かず、具体的な記憶(声の調子、会話、間)で描く。・伊良部 — 文法観察の洞察が並外れた人物 ・椎葉 — 方位語彙・狩りの語彙を持つ山人 ・都城 — (素材待ち)
2あわい 4p / 第2章後
二種類の失敗 — 「雨に降られた」と、例文倫理
成功譚(神話者列伝)の後に失敗譚を並べるコラム。言語学者の2種類の失敗を対比。
前半(1.5p)— 技術的失敗:「雨に降られた」失敗譚。迷惑の受け身が存在しない伊良部方言で、媒介言語(日本語)に引きずられて存在しない構文を文法書に載せてしまった話。/ 後半(2.5p)— 倫理的失敗:例文倫理(note 2024)。例文として「太郎が花子を殴った」「太郎が花子を殺した」のような文を使うことの問題。言語学者には抽象的な構造でも、話者・聴衆には具体的な場面と記憶が結びつく。/ 結び:「二種類の失敗を重ねてきた。これからも、おそらく重ねる」。
3あわい 3p / 第2章後
言語分析の現場
手元のデータを突き合わせて音素分析・形態音韻・品詞分類が横断的に進む、パズル的な瞬間。
「同じ語に見える2つの形が、実は別の語だった」「違う形に見えた3つの語が、音韻規則で説明できて同じ語だった」といった発見の一瞬を具体例で描く。記述言語学者の「知の悦び」を読者に見せる。伊良部島方言の実例を1つ、噛み砕いて示す。
4あわい 3p / 第3章後
kanasmunui という言葉
既発表エッセイの再録。
既発表原稿:下地理則「kanasmunuiという言葉」『ことばの波止場』Vol. 15-2(ことば研究館、国立国語研究所、2026年3月公開)。祖母が独り言のように呟いていた言葉。意味を知らないまま祖母を失った時間。言語学者になってから kanas-munu-i(「愛しいねえ」)と分析できるようになった日。切ない気持ち。処理:既発表原稿を基本的にそのまま再録。導入1行と末尾1行のみ新規書き下ろしで、前後(第3章と「あわい5」)に接続。再録元の書誌を明記。
5あわい 3p / 第3章後
言語を記録保存するとは
Boasian trilogy の実作業の手触り。
・現場での録音 — ICレコーダー、マイクの置き方、話者に緊張させない工夫 ・帰宅後の作業 — ELANによる書き起こし、文脈情報の付与 ・文法書・辞書・談話資料という3つの成果物 ・公開リポジトリへの登録 — 50年後、100年後の誰かが参照できる形に ・「記録は、言葉を話者に返す営みではない。しかし、『いない』ことにはならない」。第4章(話者の内面)への橋を作る。
6あわい 4p / 第4章後
ことばが区切る世界
語彙と文法の両面から、「言語が違えば世界の切り分け方が違う」を具体例で示す。第5章への布石。
前半(2p)— 語彙編:椎葉村のhitosubjaa(遠くから来た珍しい客に、子供がテンションを上げる様子)、東西南北で空間を把握する方言(四国、福岡、琉球列島)、椎葉の狩人の方位語彙 — 山の中で使う、東西南北でも左右でもない特殊な方位体系。/ 後半(2p)— 文法編:ウチナーヤマトグチの証拠性(シヨッタ = 直接経験 / シタ = 間接経験)、「今、雷鳴りよったよ」(聴覚)、「なんか触りよった!」(触覚)、「甘くなりよった!」(味覚)、「さっき渡ってたら、やがて轢かれよった」(反実仮想への転用、類型論的にも珍しい)。/ 結び:「ある言語が消えるとは、この区切り方が一つ消えること」。
7あわい 4p / 終章後
研究者の本棚
エッセイ型の読書案内。決定的な数冊を、人物と結びつけて短く書く。
本文(3p)— エッセイ部分:5冊を短く語る。候補:・フランツ・ボアズ — Boasian trilogyの始祖 ・ダニエル・エヴェレット『ピダハン』 — フィールド言語学者の自伝的記述 ・柳田國男『蝸牛考』 — 方言周圏論、批判対象でもあり敬意の対象でもある ・角田太作(ワロゴ語研究) — 最後の一人の話者と共に研究を続けた日本の先達 ・(素材待ち — 研究者になる原体験となった1冊)。/ 末尾(1p)— 参考文献リスト:本書で触れた文献・引用元。一般読者が次に読める本を簡潔に。
